## Installation

```
$ git clone https://github.com/rubykansai/osaka-rubykaigi-01.git
$ cd osaka-rubykaigi-01/
$ npm install
```

## Usage

```
$ npm run develop
```

## Deploy

Deployment is GitHub Pages:

$ npm run deploy
